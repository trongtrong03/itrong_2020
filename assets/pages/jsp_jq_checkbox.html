Plugin 所需文件：
jquery.js


將以下程式碼內容複製至 .html 文件內：

<input id="c1" type="checkbox" name="group[1]">
<label for="c1">Checkbox 1</label>
<input id="c2" type="checkbox" name="group[1]">
<label for="c2">Checkbox 2</label>
<input id="c3" type="checkbox" name="group[1]">
<label for="c3">Checkbox 3</label>


將以下程式碼內容複製至 .js 文件內或 <script> 元素中：

$(function() {
    $("input[type='checkbox']").on("change", function() {
        $("input[name='" + this.name + "']").not(this).prop("checked", false);
    });
});
