說明

我們可以看物件型別範例裡除了上一篇學到的函式（function）之外，尚有一項用中括號（或稱方括號 [ ]）表示的物件，它的真實身分是「陣列」（Array），不僅是 JavaScript 的內建物件，也是 JavaScript 很重要且常常需要使用到的技巧，本篇學習筆記將專注於陣列的介紹與相關操作方法。

什麼是陣列

假設今天我們要建立一本班級名冊，用 JavaScript 的話大概會是這個模樣：

var number1 = "王小明";
var number2 = "李大華";
var number3 = "張小芬";
...

在上面程式碼片段裡我們用座號當作變數名稱，依次宣告同學的順序。如果人數少可能還不覺得有什麼問題，但萬一人數很多呢？又或者未來某天來了名轉學生，勢必得需要額外宣告新的變數名稱（numberN），以在名冊裡加入新同學的資料。可是當每一筆資料都單獨通過變數命名存取，在程式操作上會相當不方便（例如變數只宣告到 number50，我就只能存取到 number50 以內的變數）。且不確定資料數量情況下，這種所有資料都以變數命名各自管理的模式也顯得不具彈性。

同一個變數，用位置分割不同資料

陣列可以讓我們統一用一個變數，以位置分割的方式，去記錄每一筆資料。拿班級名冊的例子來說，陣列的表示方法如下：

var nameList = [];
nameList[0] = "王小明";
nameList[1] = "李大華";
nameList[2] = "張小芬";

方括號 [ ] 是陣列的表示方式，資料位置的起始數字從 0 開始計算，因此你才會看到第一筆資料在陣列中的位置為 0。陣列的資料內容是可以動態新增的，假設現有名單 50 員情境下要新增第 51 人，只需要在程式碼裡新增 nameList[50] 即可（位置從 0 開始，所以第 51 人的在陣列中的第 50 個位置）。

nameList[50] = "林阿玉";

額外補充一點，[] 也可以用 new Array() 表示，兩者是一樣的：

var a = [];
var a = new Array();

除了上面的表示方式外，陣列還有更簡潔的宣告方式，即在宣告陣列變數名稱的同時，將已知的資料預先寫入陣列裡，例如：

var nameList = ["王小明", "李大華", "張小芬", ...];

陣列裡的資料以逗號 , 區隔。
