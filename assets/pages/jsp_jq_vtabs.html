Plugin 所需文件：
jquery.js


將以下程式碼內容複製至 .html 文件內：

<div class="tabs-wrap">
    <div class="tabs-nav">
        <ul>
            <li class="tabs-item">item01</li>
            <li class="tabs-item">item02</li>
            <li class="tabs-item">item03</li>
        </ul>
    </div>
    <div class="tabs-main">
        <div class="tabs-content">內容1</div>
        <div class="tabs-content">內容2</div>
        <div class="tabs-content">內容3</div>
    </div>
</div>


將以下程式碼內容複製至 .js 文件內或 <script> 元素中：

$(function(){
    $(".tabs-item:first").addClass("is-active");
    $(".tabs-content:first").addClass("is-active");
    $(".tabs-item").click(function(){
        $(this).addClass("is-active");
        $(this).siblings().removeClass("is-active");
        $(".tabs-content").siblings().removeClass("is-active");  
        $(".tabs-content").eq($(this).index()).addClass("is-active");
    });
});


僅列出此效果必要的 CSS，請將以下程式碼內容複製至 .css 文件內或 <style> 元素中：

.tabs-wrap {
    display: flex;
}

.tabs-nav {
    flex: 0 0 8rem;
}

.tabs-item {
    &.is-active {}
}

.tabs-content {
    display: none;
    &.is-active {
        display: block;
        animation-name: fadeIn;
        animation-duration: .5s;
        animation-fill-mode: forwards;
    }
}

@keyframes fadeIn {
    0% {
        opacity: 0;
    }
    100% {
        opacity: 1;
    }
}

也可以使用 float 或 display: inline-block 等方式取代 flex 使標題與內容左右排列。
