前言

條件式（If）與迭代（Iteration）於多數程式語言中是很常被使用的語法，預處理器程式語言也不例外，一起來看看如何使用。


  • 條件式（if）
  • 迭代（Iteration）

條件式（if）

我們可以用條件式定義一系列的判斷規則，標準的條件式語法由 if 與 else 組成。若條件不止兩個，則第二個之後的條件以 else if 表示，最後一樣以 else 結尾。官方文件提供幾種使用方式，以下就來練習如何定義條件式。

1. 範例一：

demo = true

if demo
    size(w, h)
        width w
        height h

div
    size(100px, 50px)

在此範例中，我們先宣告一個變數，並指定起始狀態為 true 的值給它。接著是條件式的內容，範例在條件式裡加入了一個 @mixin 函式，然後我們在 div 調用該 @mixin。最後實際看看輸出結果：

div {
    width: 100px;
    height: 50px;
}

當 demo 為 true，其條件式內容就會執行。由於一開始定義時即給了它 true 初始值，因此其條件式內容就會編譯輸出到 CSS 文件。


2. 範例二：

前一個範例是將 @mixin 寫在條件式裡，而這個範例則展示如何在 @mixin 加入條件式（但概念是一樣的）。請看程式碼：

size(w, h, use = false)
    if use
        width w
        height h

div
    size(100px, 50px, true)

編譯結果：

div {
    width: 100px;
    height: 50px;
}

這種方式的好處是讓具有條件式的 @mixin 更靈活地被使用，較彈性、非固定性的屬性可以寫在條件式裡，讓每個調用 @mixin 的選擇器單獨設定是否要輸出條件式的內容。


3. 範例三：

通常條件式除了 if 之外還有 else，表示未滿足 if 條件時，則執行 else 裡面的內容。舉例來說：

size(w, h, use = false)
    if use
        width w
        height h
    else
        width h
        height w

然後分別在不同選擇器調用：

.div-1
    size(100px, 50px, true)
  
.div-2
    size(100px, 50px)

編譯結果：

.div-1 {
    width: 100px;
    height: 50px;
}
.div-2 {
    width: 50px;
    height: 100px;
}


4. 範例四：

如果條件式需要很多個條件判斷的時候，會在 if 與 else 之間加入 else if。

font-color($color)
    if $color == Red
        color #F00

    else if $color == Blue
        color #00F

    else if $color == Green
        color #0f0

    else
        color #000

當變數 $color 填入的參數為 Red 時，color 屬性值為 #F00。若參數是 Blue，則 color 屬性值為 #00F...依此類推。假如參數不符合任一條件，則輸出 else 的內容。

調用：

.text-1
    font-color(Red)

.text-2
    font-color(Blue)

.text-3
    font-color(Green)

.text-4
    font-color(false)

編譯結果：

.text-1 {
    color: #f00;
}

.text-2 {
    color: #00f;
}

.text-3 {
    color: #0f0;
}

.text-4 {
    color: #000;
}


5. 範例五：

前面範例都在講當判定符合條件時輸出其內容，這邊要展示的則是直接定義「不符合」的條件，這種條件式通常適用在「除了某某條件外的情況」。

font-color($color)
    if (!($color == red))
        color #000
    else
        color #F00

div
    font-color(white)

此條件式用字面解釋意思就是：代入參數除了 red 之外的參數，文字顏色通通是黑色（#000）。讓我們來看輸出結果：

div {
    color: #000;
}

不過像「除非」這樣的概念，別於上面 (!(expr))，Stylus 還提供另一種語法，叫做 unless。例如：

font-color($color)        
    unless $color == red
        color #000

div
    font-color(white)

編譯結果為：

div {
    color: #000;
}


迭代（Iteration）

在 Stylus 中用來表示迭代的語法為 for ... in，同樣地，迭代有幾種手法可以運用，以下一一介紹。

1. 範例一：

看到 for，最直覺聯想到的當屬迴圈敘述了，我們可以設定一組數字範圍，透過迭代重複輸出迴圈裡的敘述句，直到變數與設定條件不符合為止。

div
    for num in 1 .. 5
        z-index num

我們先看結果：

div {
    z-index: 1;
    z-index: 2;
    z-index: 3;
    z-index: 4;
    z-index: 5;
}

這就是 for 的功用，首先在 for 後方給予一個變數名稱，然後定義其範圍，.. 指的就是「到」的意思（包含前後值本身）。條件設定完成後，編譯時就會依照條件逐一執行 for 裡的內容，直至條件不符合時結束...也就是當變數 num 值為 6 的時候。
