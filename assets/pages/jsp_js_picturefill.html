Plugin 來源：Picturefill

<picture> 是 HTML5 新增的元素標籤，可依據裡頭設定不同的媒體（media）類型，顯示相對應的影像內容，大多廣泛運用在 RWD 設計上。不過由於是 HTML5 才訂立的標準，舊版本瀏覽器普遍不支援這個標籤，而 Picturefill.js 的用途即是要讓舊瀏覽器也能識別該標籤。

Plugin 所需文件：
picturefill.js


將以下程式碼內容複製至 .html 文件內：

<picture>
    <source srcset="large.jpg" media="(min-width: 960px)">
    <source srcset="middle.jpg" media="(min-width: 768px)">
    <source srcset="small.jpg" media="(max-width: 480px)">
    <!--[if IE 9]><video style="display: none;"><![endif]-->
    <img src="default.jpg">
</picture>

引用 JS 文件：

<script src="https://cdn.rawgit.com/scottjehl/picturefill/3.0.2/dist/picturefill.js"></script>
