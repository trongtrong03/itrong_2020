基本介紹

VS Code，全名 Visual Studio Code，是一套由微軟公司開發的跨平台編輯器，與 Atom（由 GitHub 開發）相同，使用 Electron 開源框架（也是由 GitHub 開發），目前已是公布於 GitHub 開源社群裡成長迅速的開源專案之一，相較 Sublime、Atom ...等編輯器軟體，VS Code 還有內嵌 Git 版本控管的優勢，且亦能與 Gulp、Webpack 整合，相當地方便。

Stack Overflow──全世界最知名的程式設計問答之一的討論區，於 2018 的各項使用者調查裡首次加入「開發者使用環境與工具」統計項目，身為後起之秀的 VSCode 竟以 34.9% 的得票率斬獲榜首。翌年（2019）得票率更是高達 50.7%，狠狠甩開與其他開發工具之間的差距，成為當前最熱門的程式語言開發編輯器。


安裝流程

VSCode 的安裝步驟並不複雜，只需要點擊首頁的下載按鈕，或在 Download 頁中選擇符合自己工作電腦的作業系統類型與版本，下載安裝應用程式執行安裝即可。

安裝過程中建議勾選上圖用紅筆框起的兩個核選項目，它們可以幫助你迅速地透過 VSCode 開啟欲編輯的檔案。
