前言

VS Code，全名 Visual Studio Code，是一套由微軟公司開發的跨平台編輯器，與 Atom（由 GitHub 開發）相同，使用 Electron 開源框架（也是由 GitHub 開發），目前已是公布於 GitHub 開源社群裡成長迅速的開源專案之一，相較 Sublime、Atom ...等編輯器軟體，VS Code 還有內嵌 Git 版本控管的優勢，且亦能與 Gulp、Webpack 整合，相當地方便。

Stack Overflow──全世界最知名的程式設計問答之一的討論區，於 2018 的各項使用者調查裡首次加入「開發者使用環境與工具」統計項目，身為後起之秀的 VSCode 竟以 34.9% 的得票率斬獲榜首。翌年（2019）得票率更是高達 50.7%，狠狠甩開與其他開發工具之間的差距，成為當前最熱門的程式語言開發編輯器。

安裝流程

VSCode 的安裝步驟並不複雜，只需要點擊首頁的下載按鈕，或在 Download 頁中選擇符合自己工作電腦的作業系統類型與版本，下載安裝應用程式執行安裝即可。

安裝過程中建議勾選上圖用紅筆框起的兩個核選項目，它們可以幫助你迅速地透過 VSCode 開啟欲編輯的檔案。


介面介紹

左邊列表的五項功能分別是：

1. Explorer：

即檔案總管，可直接開啟本機專案，也可以輸入遠端儲存庫的 URL（例如 GitHub 上的專案），VSCode 會自動協助將遠端儲存庫的專案 clone 到指定路徑內。


2. Search：

即搜尋功能，當輸入關鍵字後，VSCode 會協助找出專案資料夾內包含其關鍵字內容的所有檔案。另外也可以利用「取代」功能，將指定文字快速取代搜尋之關鍵字。


3. Source Control：

為原始檔控制，如果你開啟之專案已有使用版本控制（例如 Git），那麼 VSCode 已經提供 Git 相關服務，不需要額外呼叫 Bash 或 GUI 執行版控。


4. Run：

這個功能可以測試開發者寫的程式腳本是否有錯誤，並提供相關 Debug 建議或方案。


5. Extensions：

即延伸模組，這個功能可說是 VSCode 的精華，所有支援 VSCode 的擴充功能或套件都可以在這個功能裡找到，幫助開發者更方便地使用 VSCode 寫程式。


以上就是 VSCode 的基本介紹，之後會再陸陸續續筆記分享一些實用的延伸模組文章，讓 VSCode 更加便利完善。最後補充一個小技巧，安裝好 VSCode 之後，你可以在命令提示字元中輸入 code 指令，就可以快速呼叫出 VSCode，若一時半刻找不到 VSCode 的捷徑，不妨可以試試看。
