前言

本篇是陣列學習筆記的下篇，將繼續聚焦於陣列操作方法的介紹。廢話不多說，馬上進入正題。


  • 複製陣列資料（copyWithin）
  • 填充陣列資料（fill）
  • 篩選陣列資料（filter）

複製陣列資料（copyWithin）

copyWithin() 方法可以複製陣列的資料，並從指定索引值的位置開始將複製的資料覆蓋原本位置的資料。它一共可填入三個參數值，分別表示：

ary.copyWithin(target, start, end);

第一個值 target 指的是要置換位置的目標索引值（必填），start 表示要從什麼位置開始複製（預設值為 0），end 則表示要在哪一個位置結束，但不包含該數值索引值本身的位置（預設為陣列長度）。剛說明定義可能仍有些懵懂，以下我們就直接來看程式碼範例：

var nameList = ["王小明", "李大華", "張小芬", "盧小小", "黃阿亮"];


只填入目標索引值：

操作陣列的資料如上，假設此時我們僅先給予 copyWithin() 一個參數值，其數值為 2：

nameList.copyWithin(2);

當 copyWithin() 內參數值只有一個數值的時候，代表陣列將從該數字對應位置的資料開始把複製的陣列資料依序取代原本的內容，由於起始值與結束值都沒有指定，因此複製的範圍涵蓋整個陣列。得到結果為下：

填充陣列資料（fill）

fill() 方法可用來將陣列中的資料填充為指定的值，其語法規則為：

ary.fill(value, start, end);

value 值是要填充的內容，start、end 和其他方法的用法相同，分別表示要從哪一個索引值位置開始執行，然後到結束值的前一個位置停止填充。

var nameList = ["王小明", "李大華", "張小芬", "盧小小", "黃阿亮"];
nameList.fill("林小玉");

像上方程式碼片段只填入 value 的話，即表示陣列所有資料通通都會被 林小玉 給填充。

See the Pen JS-learn: array 28 by Trong (@itrong) on CodePen.


篩選陣列資料（filter）

filter() 方法用來使陣列依據指定的判斷函式，依序對陣列裡每一筆資料進行判斷，並篩選出符合條件的資料。例如：

function aryFilter(n) {
    return n >= 10;
}

var nameList = [5, 1, 13, 4, 8, 55, 9, 2, 10, 24];
nameList.filter(aryFilter);

首先，我們定義了一個判斷函式 aryFilter，其內容是回傳（return）大於等於 10 的數值。接著透過 filter() 方法呼叫函式，函式便會對指定陣列裡的每一個資料執行判斷，凡是數值大於等於 10 符合條件的資料，就會被篩選出來成為新的陣列。
