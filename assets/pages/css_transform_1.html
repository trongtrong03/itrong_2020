前言

CSS3 中的 transform 屬性是一種使元素形貌產生變化的屬性，它就像 Photoshop 的「變形」功能，可以讓元素旋轉、縮放、傾斜，且變形範圍不僅侷限於二維空間，三維方向同樣可以做變化。transform 令元素運用更加靈活，也降低對外部媒介的依賴，是 CSS3 相當具代表性的屬性之一。

本篇從善如流，承襲前人的智慧，將 transform 的參數依二維（2D）與三維（3D）分別進行介紹，此為上篇。

屬性介紹

在二維空間變化中，transform 可以定義這些參數：translate、scale、rotate、skew，以及 matrix。許多專業文章內容裡，都會把 matrix（矩陣）放在前面介紹，這是因為 transform 前面四種變化方式嚴格來講都是可以用 Matrix 計算出來的用法，假如能熟悉矩陣的概念，基本就能從 transform 的課程中完滿畢業。但畢竟矩陣與數學相關，想馬上學會並非談何容易之事，因此筆者決定將矩陣另外花一個文章單獨敘述，此節先介紹其他四種參數的功用。


Translate（位移）

div {
    transform: translate(x, y);
}

位移的概念與偏移屬性（top、left...）相似，但它不需要設定 position 屬性。語法內的 x、y 值分別從元素參考點中心往水平（x 軸）與垂直（y 軸）移動。假設 translate() 裡面只有設定一個值，則視為 x 軸位移，y 軸位移為 0。

See the Pen css3: transform-1 by Trong (@itrong) on CodePen.

在範例中 x 值以百分比表示，百分比的位移程度是以元素寬度去計算，例如 50% 即位移該元素一半的寬度。

另外也可以針對 X、Y 軸單獨定義，其語法為：

div {
    transform: translateX(length);
    transform: translateY(length);
}


Scale（縮放）

div {
    transform: scale(x, y);
}

以元素參考點為中心做 X、Y 軸的倍率縮放，它填入的值只能是純數字（因為是倍數的意思），不接受其他單位，例如像素（px）或百分比（%）等。

See the Pen css3: transform-2 by Trong (@itrong) on CodePen.

假如只填入一個數字，代表 X 與 Y 軸縮放的倍率相同。此外，如同位移效果，縮放亦可分別定義 X 與 Y 軸的縮放程度：

div {
    transform: scaleX(number);
    transform: scaleY(number);
}


Rotate（旋轉）

div {
    transform: rotate(a);
}

以元素的參考點為中心進行旋轉，單位為 deg，代表角度，若填入的數字為正值，代表元素以順時針旋轉，負值則為逆時針。

See the Pen css3: transform-3 by Trong (@itrong) on CodePen.


Skew（傾斜）

div {
    transform: skew(ax, ay);
}

以元素參考點為中心，使其分別向 X、Y 軸傾斜填入的角度。倘若只填入一個值，代表只有 X 軸傾斜，Y 軸不傾斜。

See the Pen css3: transform-4 by Trong (@itrong) on CodePen.

看過前面 translate() 與 scale()，應該多少心裡有底只要能分別填入 X、Y 軸，就能更針對性的只單獨定義其中一個方向的變形屬性，因此傾斜也不例外：

div {
    transform: skewX(a);
    transform: skewY(a);
}



有些時候我們需要同時定義多個變形效果，這時只需要將各個變形參數間以「空白格」區隔開來即可。例如：

div {
    transform: translateX(-50%) rotate(45deg) skew(5deg, 15deg);
}


transform-origin

或許你心中會存在以下疑問──在上一章節介紹每個參數用途時，反反覆覆提到「以參考點為中心」，究竟什麼是參考點？要如何知道它中心點的位置？又是否能修改它？

首先我們要知道，元素進行任何變化時，都必須要以其內部的某一個點為中心，這個點就是參考點，以下拿 Photohop 當作範例：

當編輯綠色圖形變形效果時，其中央有一個圓形的小圓圈，它便是該圖形的參考點。一般而言，HTML 元素的參考點中心為正中央。假如要更改它的參考點，我們可以使用 transform-origin 屬性。

div {
    transform-origin: x y z;
}

計算起點是以元素的左上角為基準點，參數預設值為 50% 50% 0%，剛好就是元素的正中心。單位除了百分比外，也可以使用固定單位、角度（deg）或是偏移方向的名稱（top、right、bottom、left）。

See the Pen css3: transform-origin by Trong (@itrong) on CodePen.

如果 transform-origin 僅填入一個數值，則代表其三軸參考點位置皆相同；若只填入兩個值，則 Z 軸會被忽略。
