UI（使用者介面）

UI 完整名稱為 User Interface，中文翻譯為「使用者介面」，倘若要以專業術語來解釋，UI 是指系統和用戶之間進行互動和資訊交換的媒介。舉凡電腦應用軟體、手機介面、汽機車儀表板...等等，皆屬於使用者介面一部分，我們會驚訝原來 UI 在日常生活中扮演舉足輕重的角色，好的使用者介面能幫助使用者更簡單地進行操作、清楚地獲取需要資訊，或是能更輕鬆學會陌生軟體的使用方式，也因為 UI 應用是如此地頻繁，UI Designer（使用者界面設計師）成為就業市場上最搶手的職業之一。

UI 設計師的思考方向為「如何做好產品呈現」，著重具體功能與視覺安排方面的設計，UI 囊括應用範圍非常廣泛，移動裝置、遊戲、影音多媒體、廣告、工業、交通、通訊、醫療...族繁不及備載，因此，UI 設計師通常也知悉平面設計，但平面設計師卻不見得擅長 UI 設計。


UI 設計的重點：

  • 簡單且清楚。
  • 容易學習、預測、熟悉、分析。
  • 操作界面的直覺性、一致性。
  • 界面彈性與適應性 。
  • 多樣性、完善之功能。
  • 充足的操作範本及輔助。
  • 容易或不需記憶。
  • 容許使用者修改部份設定以儘量符合使用者個人的心智模型。
  • 友善性（以使用者的邏輯來思考操作方法）。
  • 容許使用者犯錯。
  • 低錯誤。
  • 整體美感。


UX（使用者經驗）

UX 完整名稱為 User Experience，中文譯為「使用者經驗」或「使用者體驗」，顧名思義，指的是使用者從特定產品或服務裡，透過人機互動獲得的五感體驗。如果說 UI 是實體產品的觀感呈現，那麼 UX 則是使用者自該項產品「無形」的使用經驗。因此，UX 設計師的工作，主要在於站在使用者的角度，思考如何幫助他們從某產品使用上得到良好感受，優化產品設計的同時，並解決操作過程可能會遇到的種種疑難雜症。


UX 設計必須思考的問題：

  • 想要展示資訊的哪些方面？
  • 想要讓使用者感受到什麼？
  • 想要讓使用者獲得什麼益處？
  • 想要讓使用者對哪些資訊印象深刻？
  • 想要令使用者思維或行動產生多少影響？
  • 如何使他們未來再次使用你的產品？


UX 設計師最好具備這些能力：

  • 同理心（換位思考）。
  • 觀察力。
  • 表達與溝通能力。
  • 耐心。
