前言

我們在開發網頁的時候，常常需要取得特定元素對象的 DOM 值，以便進行更進一步的動作。jQuery 在取得屬性值這部份也提供了許多簡化 JavaScript 寫法的方法，以下一起來看看。


  • addClass()、removeClass()
  • toggleClass()
  • hasClass()
  • html()
  • text()
  • val()
  • data()
  • attr()、removeAttr()
  • prop()

addClass()、removeClass()

看到方法名稱中含「Class」字樣，自然就知道和類別選擇器 .class 有關了。其中，.addClass() 代表添加類別，而 .removeClass() 則代表移除類別。

$("p").addClass("red-color");
// p 元素「添加」 .red-color 類別選擇器

$("p").removeClass("red-color");
// p 元素「移除」 .red-color 類別選擇器

不單只侷限添加 / 移除一個類別選擇器，它們同時也可指定多個類別選擇器進行操作，僅需每個類別選擇器之間以「空白格」區隔即可。

另外也可以在方法裡加入函式去對元素進行操作，比如：

$("p").addClass(function(n){
    return "style-" + n;
});

toggleClass()

同樣是針對類別選擇器進行動作，.toggleClass() 方法可說是 .addClass() 與 .removeClass() 的結合體，它會檢查對象元素的類別選擇器，假設當下不存在指定的 .class，則給予添加；倘若已存在該 .class，則予以移除，總而言之就像按鈕的開關，在元素上切換 .class。

$("button").click(function(){
    $("p").toggleClass("style");
});

See the Pen jq-learn-toggleClass by Trong (@itrong) on CodePen.


hasClass()

.hasClass() 主要用來檢查對象元素內是否擁有指定的 .class，較多時候會看到它運用在條件式中，依據條件判斷去執行對應的結果。

val()

前面介紹到 .html() 與 .text() 可用來設置或取得指定元素裡的內容。然而，這並不包含部份表單元素，也就是 input、select...等，如果想針對這些元素設置或取得其參數值，則需要使用 .val() 方法。

以下是取得 input 的 value 值範例：

See the Pen jq-learn: .val()-1 by Trong (@itrong) on CodePen.

當然也可以反向將參數值傳入到指定的目標表單元素中，譬如：

$("input").val("123456");


另外，經過測試，像 textarea、button 這類表單元素的值是直接寫在內容而非 value 屬性，因此可以被 .html()、.text() 取得它們元素標籤內的內容。至於像 input 或 select 裡的 option，它們的值是放在 value 屬性之中，所以無法直接用 .html()、.text() 取得。


data()

隨著 HTML5 普及，新增的全域屬性 data 也被廣泛運用在開發上。data 可以給元素標籤添加其特有的屬性名稱，以供 JavaScript 腳本交換資訊。而 jQuery 若想對 data 屬性進行操作，其語法規則為：

$(selector).data(name, value);

其中 name 代表的是 data 屬性的名稱，例如我們在 HTML 給予元素的 data 屬性為 data-color，那麼 color 就是 jQuery 要索取的 name，而 value 則是要傳入的參數。倘若只有設置 name 值，即表示要取得該 data 屬性名稱的值。例如：

attr()、removeAttr()

前面林林總總介紹了許多取得屬性的方法，那 jQuery 是否存在著一個有如萬用鑰匙般，試圖開哪一扇門就開哪一扇門──想取得元素什麼屬性就取得該屬性的方法呢？答案是有的，這個屬性叫做 .attr() 方法（也就是 Attribute 的縮寫）。不管是 .addClass() 也好，.val()、.data() 也罷，它們本質上操作的對象都是元素標籤的其中一種屬性，但無論是全域（class）、獨有（value），還是自定義（data）的，通通都可以透過 .attr() 方法取得該屬性的值，並傳入新的參數值給它。

.attr() 的語法結構與 .data() 雷同：

$(selector).attr(attribute, value);

當沒有傳入 value 參數值的時候，則表示單純要獲取該指定對象所擁有 attribute 屬性當前的值。以下我們來看看一些範例：

$('a.link').attr('href', 'https://www.google.com.tw/');
// a.link 的 href 屬性值傳入 https://www.google.com.tw/

#('h1').attr('class', 'title');
// h1 的 class 屬性值傳入 title

$('input:checkbox').attr('checked', true);
// input[type='checkbox'] 的 checked 屬性值為 true

$('li').attr('data-key', 3);
// li 的 data-key 屬性值傳入 3

需要注意的是，如果新的 value 值傳入該屬性之前已存在既定的值，原本的屬性值會直接被取代，而非附加在其後。

此外，.attr() 也可以同時設置複數個屬性的值，寫法如下：

$("img").attr({
    "src": "../images/img_1.jpg",
    "class": "large",
    "width": "300"
});


反向思考，如果要移除屬性值該怎麼做？方法有兩種，第一種是利用給予空值的方式，將原本屬性中的值給覆蓋掉，例如：

#('h1').attr('class', '');

另一種方式則是使用 .removeAttr() 方法，直接移除指定元素對象的屬性。例如：

#('h1').removeAttr('class');

See the Pen jq-learn: .removeAttr() by Trong (@itrong) on CodePen.

.removeAttr() 無法像 .removeClass() 指定一或數個值去移除元素對象中符合名稱的值，這是因為 .removeAttr() 會將指定屬性整個從整個 DOM 結構移除。


prop()

最後要介紹的是 .prop() 方法（Property），它的用法其實和 .attr() 差不多，也是可以用來取得或設置指定對象元素的屬性並賦予參數值。例如前面 .attr() 示範的例子，我們也可以透過 .prop() 設置：

$('a.link').prop('href', 'https://www.google.com.tw/');
// a.link 的 href 屬性值傳入 https://www.google.com.tw/

#('h1').prop('class', 'title');
// h1 的 class 屬性值傳入 title

$('input:checkbox').prop('checked', true);
// input[type='checkbox'] 的 checked 屬性值為 true

$('li').prop('data-key', 3);
// li 的 data-key 屬性值傳入 3

既然 jQuery 創造了這兩種性質這麼相近的方法，.attr() 與 .prop() 的差別究竟在哪裡呢？答案在於各自回傳值的資料型別。前者返回的值資料型別結果是字串，因為其取值是基於 HTML 上的元素的屬性，工作原理相當於 JavaScript 的 setAttribute() 方法，會將值轉換成字串。後者 .prop() 則是對我們用 JacaScript 或 jQuery 獲取到的 DOM 元素屬性物件取值，屬性物件中有多少屬性，就能獲取多少值，回傳的結果可以是任何資料型別，像是字串、數字或布林值。

以下我們用 input 的 checkbox 做範例，就能明顯區別出兩個方法回傳值的不同：

<input type="checkbox" class="c1" checked="true" />
<input type="checkbox" class="c2" checked="false" />
<input type="checkbox" class="c3" />

我們設置了三個 checkbox，並分別設置 checked 屬性為 true、false，以及不設置。然後接著用 .attr()、.prop() 方法去檢索它們：

$(".c1").attr(checked);    // 回傳的值為 checked
$(".c1").prop(checked);    // 回傳的值為 true

$(".c2").attr(checked);    // 回傳的值為 checked
$(".c2").prop(checked);    // 回傳的值為 true

$(".c3").attr(checked);    // 回傳的值為 undefined
$(".c3").prop(checked);    // 回傳的值為 false

.attr() 方法會去取指定對象的 checked 屬性，若有則回傳字串，若無則返回 undefined。相反地，.prop() 取該元素屬性的 checked 值，返回 true 或 false 的結果。
