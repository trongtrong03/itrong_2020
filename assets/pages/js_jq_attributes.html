前言

我們在開發網頁的時候，常常需要取得特定元素對象的 DOM 值，以便進行更進一步的動作。jQuery 在取得屬性值這部份也提供了許多簡化 JavaScript 寫法的方法，以下一起來看看。


  • addClass()、removeClass()
  • toggleClass()
  • hasClass()
  • html()
  • text()
  • val()
  • attr()、removeAttr()
  • prop()

addClass()、removeClass()

看到方法名稱中含「Class」字樣，自然就知道和類別選擇器 .class 有關了。其中，.addClass() 代表添加類別，而 .removeClass() 則代表移除類別。

$("p").addClass("red-color");
// p 元素「添加」 .red-color 類別選擇器

$("p").removeClass("red-color");
// p 元素「移除」 .red-color 類別選擇器

不單只侷限添加 / 移除一個類別選擇器，它們同時也可指定多個類別選擇器進行操作，僅需每個類別選擇器之間以「空白格」區隔即可。

另外也可以在方法裡加入函式去對元素進行操作，比如：

$("p").addClass(function(n){
    return "style-" + n;
});

toggleClass()

同樣是針對類別選擇器進行動作，.toggleClass() 方法可說是 .addClass() 與 .removeClass() 的結合體，它會檢查對象元素的類別選擇器，假設當下不存在指定的 .class，則給予添加；倘若已存在該 .class，則予以移除，總而言之就像按鈕的開關，在元素上切換 .class。

$("button").click(function(){
    $("p").toggleClass("style");
});

See the Pen jq-learn-toggleClass by Trong (@itrong) on CodePen.


hasClass()

.hasClass() 主要用來檢查對象元素內是否擁有指定的 .class，較多時候會看到它運用在條件式中，依據條件判斷去執行對應的結果。

See the Pen jq-learn-hasClass by Trong (@itrong) on CodePen.

text()

.text() 與 .html() 雷同，差別在於前者只能設置或取得指定元素內的純文字，假如對象元素內包含其他元素標籤，則會直接忽視。

See the Pen js-learn-text by Trong (@itrong) on CodePen.


val()

前面介紹到 .html() 與 .text() 可用來設置或取得指定元素裡的內容。然而，這並不包含部份表單元素，也就是 input、select...等，如果想針對這些元素設置或取得其參數值，則需要使用 .val() 方法。
