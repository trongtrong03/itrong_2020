Plugin 來源：Masonry

Plugin 所需文件：
jquery.js
masonry.js


將以下程式碼內容複製至 .html 文件內：

<div class="grid-wrap">
    <div class="grid-item">
        <img src="img01">
    </div>
    <div class="grid-item">
        <img src="img02">
    </div>
    <div class="grid-item">
        <img src="img...">
    </div>
</div>

引用 JS 文件：

<script src="https://cdn.jsdelivr.net/jquery.mixitup/latest/jquery.mixitup.min.js"></script>


將以下程式碼內容複製至 .js 文件內或 <script> 元素中：

$(function() {
    $('.grid-wrap').masonry({
        itemSelector: '.grid-item',
        // columnWidth: 160,
        gutter: 8       // 項目間距
    });
    $('.grid-item').each(function(i){
        setTimeout(function(){
            $('.grid-item').eq(i).addClass('is-visible');
        }, 200 * i);
    });
});


僅列出此效果必要的 CSS，請將以下程式碼內容複製至 .css 文件內或 <style> 元素中：

.grid-item {
    opacity: 0;
    &.is-visible {
        animation: Pop-in 600ms ease-in forwards;
    }
}

@keyframes Pop-in {
    0% {
        opacity: 0;
        transform: scale(0.3) rotateY(90deg);
    }
    60% {
        transform: scale(1.4);
    }
    90% {
        transform: scale(0.9);
    }
    100% {
        opacity: 1;
    }
}
