Plugin 來源：dotdotdot

Plugin 所需文件：
jquery.js
jquery.dotdotdot.js

jQuery.dotdotdot 提供多個不同版本，請依據自身使用的 jQuery 版本引用相對應的腳本文件即可。


將以下程式碼內容複製至 .html 文件內：

<div class="dot-text">
    <p>文字內容</p>
</div>


將以下程式碼內容複製至 .js 文件內或 <script> 元素中：

$(function() {
    $(".dot-text").dotdotdot({
        watch: "window"
    });
});
